# TRIFONCTION

Trifonction est un site web rassemblant tous les outils nécessaires pour apprendre le plus rapidement et convenablement possible le triathlon dans tous ses aspects. 
Avec un forum pour chacun des sports , et un plan d'entrainement établit non pas seulement avec les performances de l'athlète, mais aussi avec son état de santé actuel de manière à avoir un suivi le plus optimisé et réaliste
possible. De plus , Trifonction possède une page dédié pour la science afin de comprendre toute l'anatomie du corps humain pendant l'effort et analyser comme celui-ci fonctionne et ainsi mieux comprendre comment vous entrainez.
